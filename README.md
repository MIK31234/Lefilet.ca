# Lefilet.ca
Le Filet est un restaurant principalement dédié aux produits de la mer. Les restaurateurs Claude Pelletier et Hubert Marsolais, appuyés par le chef Yasu Okazaki, vous accueillent dans leur établissement situé face au Mont-Royal. Une terrasse accueillante est aussi installée directement face aux terrains de tennis du Parc Jeanne-Mance
